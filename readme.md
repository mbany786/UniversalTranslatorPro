 # Universal PDF & Webpage Translator\
\
An AI-powered tool that converts PDFs (via OCR) and Webpages into polished, academic-grade English translations using OpenAI's GPT-4. Specialized for spiritual, philosophical, and metaphysical texts.\
\
## Features\
- **PDF OCR:** Extracts text from images/scanned PDFs using EasyOCR (Urdu, Arabic, English support).\
- **Webpage Scraping:** Cleanly extracts main content from URLs for translation.\
- **Context-Aware AI:** Uses custom system prompts to handle complex theological and Sufi terminology.\
- **Multi-Format Export:** Saves translated content as `.txt`, `.docx`, and `.pdf`.\
- **Progress Tracking:** Saves state during large PDF translations to allow resuming.\
\
## Prerequisites\
You must have `poppler` installed on your system to handle PDF-to-image conversion:\
- **Mac:** `brew install poppler`\
- **Windows:** Download from [poppler for Windows](https://github.com/oschwartz10612/poppler-windows/releases/).\
